## 🎯 Overview

**SHEHROZ SOFT** (in partnership with Apex Software Labs) is a research-driven computer science and software engineering consultancy specializing in:

✨ **Three Core Disciplines:**
1. **Computer Science Research** - Algorithmic optimization, distributed consensus, formal verification
2. **Software Engineering** - Scalable systems, microservices, cloud architecture
3. **Systems Aesthetics** - Beautiful, responsive interfaces for complex technical systems

This application serves as an interactive portfolio and technical consulting platform featuring:
- Real-time architectural blueprint compilation powered by Google Gemini AI
- Interactive distributed systems visualization tools
- Live technical consultation chatbot
- Comprehensive case studies and portfolio showcase

## ✨ Features

### 🏗️ Interactive Architecture Studio
- **System Blueprint Compiler**: Input technical requirements and receive AI-generated system architectures
- **Topology Diagrams**: ASCII art representations of distributed system layouts
- **Risk Analysis**: Automated identification of engineering bottlenecks
- **Component Ledger**: Detailed breakdown of system components and their roles
- **Pre-configured Presets**: 3 production-grade example architectures (Fintech, IoT, Video Analytics)

### 🔄 Consistent Hashing Ring Simulator
- **Visual Ring Topology**: Interactive 360° hash ring with nodes and data keys
- **Dynamic Node Management**: Add/remove nodes and observe key reallocation
- **Caching Visualization**: See exactly which nodes store which data
- **Elastic Scaling**: Demonstrate how consistent hashing limits cache misses to 1/N
- **Real-time Mapping**: Live routing table showing key-to-node assignments

### 🤖 AI Systems Consultant
- **Technical Q&A**: Ask questions about distributed systems, databases, and architecture
- **Multi-turn Conversations**: Maintain context across multiple exchanges
- **Preset Queries**: Quick access to popular technical topics
  - SQL vs NoSQL scaling strategies
  - Kafka vs RabbitMQ comparisons
  - Thundering Herd problem solutions
  - Distributed transaction patterns

### 📚 Portfolio & Case Studies
4 detailed case studies showcasing real-world projects:
1. **Veridian Ledger Core** - Byzantine-fault tolerant distributed ledger
2. **Helios Telemetry Mesh** - IoT ingestion pipeline (5.4M events/sec)
3. **Cortex Diagnostic Canvas** - Healthcare UI optimization
4. **Prism Global Edge Cache** - Multi-region consistent hashing

### 🎨 Design Philosophy
- **User-Centered Design**: Complex systems made intuitive
- **Functional Aesthetics**: Form follows technical framework
- **High Fidelity UI**: Dark cosmic canvas with professional typography

## 🛠️ Tech Stack

### Frontend
- **React 19** - UI framework with concurrent rendering
- **TypeScript** - Type-safe development
- **Tailwind CSS 4** - Utility-first styling with Vite integration
- **motion/react** - Smooth animations and transitions
- **Lucide React** - Beautiful icon library
- **Vite 6** - Lightning-fast build tool

### Backend
- **Express.js** - Node.js server framework
- **Google Gemini API** - Generative AI for system architecture design and consulting
- **dotenv** - Environment variable management

### DevOps & Build
- **esbuild** - Fast JavaScript bundler
- **tsx** - TypeScript execution for Node.js
- **Vite SSR/SPA** - Server-side middleware and static site generation
